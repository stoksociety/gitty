About Git_App

Search GitHub

This is a search app that aims to implement API request service and search through github users' data. To get information about a user from this app, input their github username and click search. You should be able to view all their public repositories. Also click on their name to view full repo details on the github website. Visit the homepage, fill the form(with correct username) to make a github search.
